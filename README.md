# App Store Website Template

A modern, mobile-responsive static website template ready for deployment on GitHub Pages.

## Features

- **Mobile-First Design**: Fully responsive layout that works on all devices
- **Modern HTML5**: Uses semantic elements for better structure
- **Clean CSS**: Single stylesheet with Flexbox/Grid layouts
- **Two Pages**: Home page with hero section and Privacy Policy page
- **Flexible Logo**: Support for both text and image logos
- **GitHub Pages Ready**: Deploy directly to GitHub Pages

## Pages

- **Home** (`index.html`): Features a two-column hero section with call-to-action buttons
- **Privacy** (`privacy.html`): Comprehensive privacy policy page

## Files Structure

```
├── index.html          # Home page
├── privacy.html        # Privacy policy page
├── style.css           # Main stylesheet
├── data.txt            # Static text file (accessible via URL)
└── README.md           # This file
```

## Customization

### Logo Options

The template supports both text and image logos. To switch between them:

1. Open `index.html` and `privacy.html`
2. Follow the comments in the navigation section to choose your logo type
3. For image logos, place your logo file in the root directory and update the path

### Colors and Styling

All customization points are marked with comments in `style.css`. Look for `/* Customize: */` comments to easily modify:
- Colors
- Font sizes
- Spacing
- Layout dimensions

## Deployment

### GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select the `main` branch as the source
4. Your site will be live at `https://yourusername.github.io/repository-name/`
